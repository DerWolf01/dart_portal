Factory method to create a Portal instance based on a portal implementation.

This method uses reflection to find and return the first Portal annotation associated with the provided portal class. It is useful for dynamically obtaining portal metadata at runtime.

Implementation

factory Portal.fromInstance(dynamic portal) {
  return metadata(portal).whereType<Portal>().first;
}
